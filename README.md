# HTML Learning:

Repositório dedicado aos meus estudos e práticas de HTML5, com foco na construção de uma base sólida para o desenvolvimento front-end e para a futura aplicação de CSS e JavaScript.

## Conteúdos estudados:

- Estrutura básica de documentos HTML
- Cabeçalhos (`h1` a `h6`)
- Parágrafos e formatação de texto
- Links e navegação
- Imagens
- Listas ordenadas e não ordenadas
- Tabelas
- Formulários
- Comentários e boas práticas
- Organização de arquivos e pastas

##  Objetivo:

Compreender a importância da semântica e da estruturação correta do HTML, criando páginas bem organizadas e acessíveis, que servirão como base para estilização com CSS e interatividade com JavaScript.
